# Тема 3: [Применение условных вероятностей в анализе данных](ConditionalProbabilitiesInDataAnalysis.py)
На наборе данных '[Amazon Product Reviews](https://www.kaggle.com/datasets/arhamrumi/amazon-product-reviews)' демонстрируется применение условных вероятностей.\
Найти:
1) Какова вероятность, что отзыв хороший, если отзыв длинный?
2) Какова вероятность, что отзыв хороший, если отзыв короткий?
3) Какова вероятность, что отзыв полезный, если он хороший?

**Описание набора данных:**\
*Используемые данные:*
1) HelpfulnessNumerator - количество голосов полезно(столько пользователей сочли этот отзыв полезным);
2) HelpfulnessDenominator - общее количество голосов(столько пользователей отреагировали на этот отзыв, тоесть сочли этот отзыв полезным или бесполезным);
3) Score - оценка, которую поставил пользователь товару, когда оставлял свой отзыв;
4) Text - текст отзыва.

*Неиспользуемые данные:*
1) id - id записи
2) ProductId - id продукта;
3) UserId - id пользователя;
4) ProfileName - Имя пользователя;
5) Time - время;
6) Summary - обобщенный отзыв(кратко).

**Примечание:**
Хорошим отзывом считается отзыв с оценкой(Score) >= 4, плохим с оценкой(Score) < 4.\
Для каждого задание приведено 2 варианта решения.\
Первый вариант - использована формула.\
Второй вариант - использован фильтр.\
***Важно:*** После скачивания файла с данными и его распаковки, для корректной работы программы файл "**Reviews.csv**" необходимо переименовать в "**data.csv**".
