# Задачи для Github по предмету "Теория вероятностей"
# Тема 1: [Моделирование геометрической вероятности](GeometricProbabilityModeling.py)
Смоделируйте вычисление геометрической вероятности. Дана сторона квадрата, в который вписана окружность с радиусом radius. Эта окружность описана вокруг малого квадрата. Требуется определить вероятность попадания дротиков в круг, а также вероятность попадания дротиков в малый квадрат. Показать теоретический расчет и графическую иллюстрацию для рассматриваемого примера.
