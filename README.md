# Задачи для Github по предмету "Теория вероятностей"
# Тема 1: [Моделирование геометрической вероятности](GeometricProbabilityModeling.py)
Смоделируйте вычисление геометрической вероятности. Дана сторона квадрата, в который вписана окружность с радиусом radius. Эта окружность описана вокруг малого квадрата. Требуется определить вероятность попадания дротиков в круг, а также вероятность попадания дротиков в малый квадрат. Показать теоретический расчет и графическую иллюстрацию для рассматриваемого примера.

# Тема 2: [Моделирование связи частотного и классического определения вероятности](ModelingFrequencyAndClassical.py)
Приведите пример, показывающий связь частотного и классического определения вероятности с помощью моделирования на примере задачи про красно-синие шары. Графически показать стремление частотной вероятности к классической при увеличении числа опытов.

**Примечание к использованию программы.**

При вводе с клавиатуры:
*на вход подается количество крассных шаров; количество синих шаров; количество вынимаемых красных шаров; количество вынимаемых синих шаров.*

При заполнении случайными значениями:
*аргументы не вводятся*
