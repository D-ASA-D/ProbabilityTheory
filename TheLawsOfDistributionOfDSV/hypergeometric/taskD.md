# Задача:
На сервер ежедневно поступает 60 HTTP запросов, из них 20 вредоносных. Все запросы сохраняются в журнал логов. Ежедневно случайным образом отбирается 13 HTTP запросов для проверки. Случайная величина X - число вредоносных запросов в выборке за день.
Составить закон распределения числа вредоносных запросов в выборке, найти математическое ожидание, дисперсию, среднее квадратическое отклонение и моду, отобразить многоугольник распределения.\
